
The shell is a program that takes keyboard commands and passes them to the operating system to carry out.

| Command | What it does               |
| ------- | -------------------------- |
| `date`  | Current date/time          |
| `cal`   | Calendar for current month |
| `df`    | Disk free space            |
| `free`  | Free memory (RAM + swap)   |
| `exit`  | End session                |

| Shortcut | Action |
|----------|--------|
| `Ctrl-D` | End session (same as `exit`) |
| `Ctrl-Alt-F1..F6` | Switch to virtual consoles |
| `Alt-F7` | Return to GUI desktop |

- Shell prompt: `$` = normal user, `#` = root
- `bash` = Bourne-Again Shell
- Don't use `Ctrl-C`/`Ctrl-V` in terminal -- they mean different things
