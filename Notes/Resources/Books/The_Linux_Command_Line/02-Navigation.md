
|Command | What it does |
|---------|-------------|
| `pwd` | Print working directory |
| `cd` | Change directory |
| `ls` | List directory contents |

### cd shortcuts

| Shortcut   | Goes to             |
| ---------- | ------------------- |
| `cd`       | Home dir            |
| `cd -`     | Previous dir        |
| `cd ~user` | That user's home    |
| cd ~       | Current user's home |

### Path types
- **Absolute**: starts with `/` -- e.g. `/usr/bin`
- **Relative**: from current dir -- uses `.` (here) and `..` (parent)
- Files starting with `.` are **hidden** (`ls -a` to see)
- Filenames are **case sensitive**
