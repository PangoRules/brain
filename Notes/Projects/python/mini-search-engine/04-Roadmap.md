# High-level roadmap

Your mini search engine can be built in **7 phases**:

1. **Project foundations**
2. **Text processing + local document search**
3. **Web crawling**
4. **Indexing**
5. **Ranking**
6. **Search API + pagination**
7. **Polish for resume quality**

The reason for this order is simple:

- first learn how search works on **documents you already have**
- then learn how to **collect** documents from the web
- then learn how to **rank**
- then wrap it with an API and UI
- then polish it into something worth showing
